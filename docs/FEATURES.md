# EDITH Feature Documentation 🌟

Here is a functional catalog of features integrated into EDITH:

## 1. Academic Manager
- **Timetables**: Reminds Boss of classrooms and start times.
- **Assignments**: Tracks deadline durations and calculates remaining days.
- **Study Mode**: Initiates Pomodoro sessions, plays focus binaural loops, and blocks browser domains.
- **Research Helper**: Scrapes arXiv abstracts for given queries and returns concept summaries.

## 2. Hostel Manager
- **Curfew Alerts**: Counts down hours until the gate locks.
- **Mess Reminders**: Checks meal times.
- **Power Saver**: Suspends high-energy threads (camera/speech listeners) and dims screens during outages.

## 3. Student Budget & Deals
- **Expenses**: Logs items and auto-parses simulated banking SMS messages.
- **Deals Finder**: Scrapes prices online to find the cheapest options.

## 4. Emergency Fail-safes
- **SOS Dispatch**: Collects IP geolocation maps and broadcasts panic messages to emergency contacts.
- **Dead Man Switch**: Fires alert when Boss is unresponsive for configured hours.
- **Fake Call**: rings simulated phone calls on schedules.
